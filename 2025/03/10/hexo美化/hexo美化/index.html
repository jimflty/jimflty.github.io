Hexo使用学习笔记

Hexo建站完毕，还需经过一定的美化达到观感上的满意。Hexo可进行美化的部分包括主题、背景，以及文档排版（layout）。Hexo初始主题已经非常简洁美观，故本文主要进行自定义背景、文档排版美化等，排除在美化过程中发现的问题，查看最终美化结果。

自定义背景

根据Hexo 自定义本地封面图的办法 | Juice’s Blog中提到的方法：

  1. 图片的http链接

  2. 把图片放在博客根目录/source/images

  3. 通过 abbrlink 和 assetimg 的方式（需要安装abbrlink&&asset-img）

三种方式均可实现自定义背景，但考虑到http有内容替换的可能性，把心仪的图片保存到本地的方式更为稳妥且不用额外安装插件，因此选择采用2。

由于距该教程已过去两年，Hexo文件结构已然发生变化，现在放置背景图片的目标文件夹为根目录/public/css/images

我们将图片放置到目标文件夹并命名为banner.jpg，即可完成背景替换。

文档排版

layout

layout有助于更好地控制博客页面的结构和样式，使我们能够根据需求轻松创建和管理不同类型的页面和文章。详细教程：Hexo 教程 (四) Layout 详解：从原理到应用 | 致以无暇之人

layout在官网文档中的另一种说法是布局，默认有三种：

  1. post（页面仍然为Hexo主题）
  2. page（新建页面）
  3. drafts（草稿，不会在blog中显示，后续可通过publish命令发布为post）

Front-matter

Front-matter用于post在博客页面的显示和分类，常用属性有：

设置 描述 默认值
title 标题 文章的文件名
date 建立日期 文件建立日期
tags 标签
categories 分类
permalink 覆盖文章的永久链接. 永久链接应该以 / 或 .html 结尾 null
excerpt 设置文章在博客页面显示的摘要，默认显示全文
