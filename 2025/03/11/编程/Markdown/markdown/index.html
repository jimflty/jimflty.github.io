Markdown学习笔记

2025/3/11 编程 Markdown

Hexo默认使用Markdown语言进行页面渲染，为使写作更加流畅、排版更符合阅读习惯，有必要学习Markdown语言。

Markdown介绍

​ Markdown是一种轻量级标记语言，排版语法简洁，让人们更多地关注内容本身而非排版。它使用易读易写的纯文本格式编写文档，可与HTML混编，可导出 HTML、PDF 以及本身的 .md 格式的文件。

Markdown语法

标题

通过#来标记标题，#的数量表示标题等级，最高可支持六级标题。

Markdown语法：# 一级标题

实际效果：

一级标题

段落

使用空白行对文本进行分隔（在两行文本中新增一行空白行），使文本段落结构更清晰。

语法及效果：

段落1
段落2（1、2之间无空行）

段落3（2、3之间有空行）

换行

在一行的末尾添加至少两个空格或<br>，然后按回车键,即可创建一个换行。

强调（加粗）

在想要加粗强调的文本两边加上两个星号**，即可将文本标粗。

语法：**需要强调的文本**

效果：需要强调的文本

引用

通过在文本前加上符号>创建引用。

语法：> 引用文本

效果：

引用文本

序列

可通过在前面添加数字.形式创建有序序列，数字可任意，但应以1开头。在前面添加-创建无
序序列。

语法：

1. xxx
1. xxx
1. xxx

效果：

  1. xxx
  2. xxx
  3. xxx

语法：

- xxx
- yyy
- zzz

效果：

  • xxx
  • yyy
  • zzz

代码

要将单词或短语表示为代码，将其包裹在反引号 (`) 中。

语法：（`文本`）

效果：文本

要创建代码块，可通过在代码块前缩进一个制表符或四个空格以上，或用```/~~~将代码块框出。可在反引号后面添加代码对应的语言，如c++、python等。
效果：

xxx
    yyy
zzz

分割线

用三个星号（*）或破折号（-）或下划线（_）创建分割线，并且不能包含其他内容

URL链接

链接文本放在中括号内，链接地址放在后面的括号中，链接title可选。

语法：[超链接显示名](超链接地址 "超链接title")

图片

要添加图像，使用感叹号 (!)， 然后在方括号增加替代文本，图片链接放在圆括号里，括号里的链接后可以增加一个可选的图片标题文本。

语法：![图片alt](图片链接 "图片title")

效果：

转义字符

在误标的文本前加反斜杠\，让其显示为原本的形式。

对于想要添加星号但不想标粗的文本：
\*xxx

效果：
*xxx

在 HTML 文件中，有两个字符需要特殊处理： < 和 & 。 < 符号用于起始标签，& 符号则用于标记 HTML 实体，如果你只是想要使用这些符号，你必须要使用实体的形式，像是 < 和 &。

表格

添加表格，使用三个或多个连字符（—）创建每列的标题，并使用管道（|）分隔每列。可以选择在表的任一端添加管道。

语法：

| Syntax      | Description |
| ----------- | ----------- |
| Header      | Title       |
| Paragraph   | Text        |

效果：

Syntax Description
Header Title
Paragraph Text
