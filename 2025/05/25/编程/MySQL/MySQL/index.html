MySQL学习笔记

2025/5/25 编程 MySQL

MySQL介绍

MySQL 是最流行的关系型数据库管理系统，在 WEB 应用方面 MySQL 是最好的 RDBMS(Relational Database Management System：关系数据库管理系统)应用软件之一。本文介绍MySQL的安装以及创建数据库、数据操作等基本SQL语法。

MySQL安装

从官网下载对应平台的安装包，一路点击Next即可完成，这边不再赘述。

MySQL Shell

MySQL Shell作为控制终端，用于执行大部分操作。MySQL也为VSCODE推出了专插件MySQL Shell for VSCode，使Shell可在VSCODE上运行。VSCODE具有平台集成功能强大、扩展众多等优点，因此选择使用该插件操作数据库。

下载插件后，左侧快捷栏会增加MySQL Shell入口，进入后即可连接DATABASE。

SQL基础

关系型数据库系统RDBMS

RDBMS(Relational Database Management System)采用关系模型组织数据，借助集合代数等数学概念处理数据库中的数据。

SQL

SQL(Structrued Query Language)结构化查询语言，用于操作关系型数据库。

基本操作 语法
连接数据库 mysql -u用户名 -p
数据库操作 语法
创建数据库 CREATE DATABASE数据库名
查看数据库 SHOW DATABASES
使用数据库 USE数据库名
删除数据库 DROP DATABSE数据库名
数据表操作 语法
创建表 CREATE TABLE表名
查看表 SHOW TABLES
查看表结构 DESC表名
删除表 DROP TABLE表名
数据操作 语法
插入数据 INSERT INTO表名(列1，列2)
VALUES (值1,值2)
查询数据 SELECT * FROM 表名
SELECT 列1，列2 FROM 表名 WHERE 条件语句
更新数据 UPDATE 表名 SET 列1 = n WHERE 条件语句
删除数据 DELETE FROM 表名 WHERE 条件语句
条件语句 语法
比较=,<,> 列名 {=,<,>} 值
约束(添加在创建表语句后) 语法
PRIMARY KEY 主键，唯一且非空
AUTO_INCREMENT 自动增长
NOT NULL 不允许为空
UNIQUE 唯一值
DEFAULT 默认值
索引(添加在查询数据后) 语法
排序 SELECT * FROM 表名 ORDER BY 列名 DESC
添加索引 CREATE INDEX 索引名 ON 表名(列名)
